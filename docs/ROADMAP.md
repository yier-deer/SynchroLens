# SynchroLens 功能路线图

> **SynchroLens** — AI 同声传译助手，实时捕获音频、语音转写、流式翻译、字幕渲染，让跨语言沟通零障碍。

---

## v0.1 — MVP（72 小时交付）

> 目标：端到端跑通核心链路，从音频捕获到字幕呈现，形成可用最小产品。

### P0 — 核心链路（必须完成）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 系统音频捕获 | 捕获系统输出音频（如会议、视频音轨） | P0 |
| 麦克风捕获 | 捕获麦克风输入音频（如现场发言） | P0 |
| 实时 STT | 接入讯飞语音识别，流式返回转写文本 | P0 |
| 流式翻译 | 接入 DeepSeek，对 STT 输出进行逐句翻译 | P0 |
| 字幕渲染 | 在主界面实时展示原文与译文 | P0 |
| 当前句纠正 | 用户可对当前正在显示的句子进行即时修正 | P0 |
| 悬浮字幕窗口 | 独立置顶悬浮窗，始终显示最新字幕内容 | P0 |
| 控制悬浮窗 | 小型悬浮控制面板，支持开始/暂停/停止等操作 | P0 |

### P1 — 增值功能（尽量完成）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 笔记自动写入（Markdown） | 将转写与翻译内容自动保存为 Markdown 文件 | P1 |
| 笔记纠正标注 | 在笔记中标注用户手动纠正的内容，区分原始输出与修正 | P1 |
| 自动总结 | 会话结束后自动生成内容摘要 | P1 |

### P2 — 扩展功能（时间允许则做）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 本地文件音频 | 支持导入本地音频/视频文件进行离线转写与翻译 | P2 |

## v0.2 — 功能完善

> 目标：补齐体验短板，增加智能辅助能力，从"能用"迈向"好用"。

| 功能 | 说明 | 优先级 | 预计阶段 |
|------|------|--------|----------|
| 语音播报（TTS） | 将翻译结果通过 TTS 实时朗读，支持语速/音色选择 | P1 | v0.2 |
| 日志查看器 — 阅读视图 | 以阅读模式浏览历史会话记录 | P1 | v0.2 |
| 日志查看器 — 编辑视图 | 以编辑模式修改历史记录内容 | P2 | v0.2 |
| 日志查看器 — 右键改进对话框 | 右键选中文本弹出改进对话框，支持 AI 辅助润色 | P2 | v0.2 |
| 用户词典学习 | 从用户纠正记录中学习偏好，自动优化后续转写/翻译 | P1 | v0.2 |
| 领域偏好适配 | 记录用户常看领域，自动加载对应术语库 | P2 | v0.2 |

## v1.0 — 正式版

> 目标：产品成熟，体验流畅，具备个性化与可扩展能力。

| 功能 | 说明 | 优先级 | 预计阶段 |
|------|------|--------|----------|
| 多语言 STT 支持 | 除中文外支持英语、日语等多语种实时识别 | P1 | v1.0 |
| 多翻译引擎切换 | 支持在 DeepSeek / Google / DeepL 等引擎间切换 | P1 | v1.0 |
| 自定义术语库 | 用户可手动维护术语表，翻译时优先匹配 | P1 | v1.0 |
| 会话管理 | 按日期/标签归档历史会话，支持搜索与导出 | P1 | v1.0 |
| 插件系统 | 开放插件接口，支持第三方扩展（如 OCR、录屏等） | P2 | v1.0 |
| 跨平台支持 | Windows / macOS / Linux 全平台覆盖 | P1 | v1.0 |
| 性能与稳定性 | 长时间运行内存稳定、低延迟、断线重连 | P0 | v1.0 |
| 无障碍适配 | 键盘快捷键、高对比度模式、屏幕阅读器支持 | P2 | v1.0 |

## 路线图总览

```
v0.1 (MVP/72h)
├── P0: 音频捕获 → STT → 翻译 → 字幕渲染 → 悬浮窗 → 纠正
├── P1: 笔记写入 → 纠正标注 → 自动总结
└── P2: 本地文件音频

v0.2 (完善)
├── TTS 语音播报
├── 日志查看器（阅读/编辑/右键改进）
├── 用户词典学习
└── 领域偏好适配

v1.0 (正式版)
├── 多语言 STT / 多翻译引擎
├── 自定义术语库 / 会话管理
├── 插件系统 / 跨平台
├── 性能稳定性 / 无障碍
└── ...
```
